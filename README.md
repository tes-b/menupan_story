# menupan_story (메뉴판스토리)

<http://3.38.115.215:5000/>  
  
지역정보를 활용해 외식업 관련 통계를 제공하는 어플리케이션.  

구현된 기능  
- 행정구역 검색  
- 행정구역 지도 표시 (naver map)  
- 검색된 행정구역의 관련 통계 차트 (plotly)  

활용 기술과 라이브러리  
- flask     : 어플리케이션 제작  
- bootstrap : 어플리케이션 템플릿  
- plotly    : 차트 생성  
- mysql     : 데이터베이스 
- AWS EC2   : 어플리케이션 배포
- naver-map-api : 지도 표시
